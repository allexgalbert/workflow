Win+R gpedit.msc

Отключить Windows Defender
Computer Configuration->Administrative Templates->Windows Components->Windows Defender Antivirus
- Turn off Windows - в Enabled

Диспетчер задач вкладка Startup
- Windows Defender в Disabled

Отключить Windows Search
Computer Configuration->Administrative Templates->Windows Components > Search
- Do not allow web search - в Enabled
- Don’t search the web or display web results in Search - в Enabled
- Don’t search the web or display web result in Search over metered connections - в Enabled

Отключить Windows Telemetry
Computer Configuration->Administrative Templates->Windows Components->Data Collection and Preview Builds
- Allow Telemetry - в Disabled

Отключить Windows OneDrive
Computer Configuration->Administrative Templates->Windows Components->OneDrive
- Prevent the usage of OneDrive for file storage - в Enabled

Отключить Windows lock screen
Computer Configuration->Administrative Templates->Control Panel->Personalization
- Do not display the lock screen - в Enabled

Win+R regedit

Включить Windows Photo Viewer
HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows Photo Viewer\Capabilities\FileAssociations
создать строковые параметры
- ключ ".png" значение "PhotoViewer.FileAssoc.Tiff"
- ключ ".jpg" значение "PhotoViewer.FileAssoc.Tiff"
- ключ ".jpeg" значение "PhotoViewer.FileAssoc.Tiff"

Отключить эскизы окон на панели задач
HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Explorer\Advanced
- создать DWORD-параметр, ключ ExtendedUIHoverTime значение 60000

Отключить стрелки на ярлыках
HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\Explorer
- клик на Explorer, New->Key, назвать Shell Icons
- внутри папки, New->String Value, назвать 29
- если пропали иконки после восстановления бекапа, то пересоздать

Удалить библиотеки из Проводника
HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\Explorer\MyComputer\NameSpace
- удалить все цифробуквенные папки

По Alt+Tab только иконки
HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Explorer
- AltTabSettings DWORD 0 = Thumbnails, 1 = Classic icons
