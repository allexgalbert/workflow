Ускорение

<link rel='preload' href='css.css' as='style'> загрузить и кэшировать ресурс (js,css) для использования через несколько секунд
<link rel='prefetch' href='css.css' as='style'> загрузить и кэшировать ресурс (js,css) для следующей страницы
as: style, script, font

<link rel='preconnect' href='https://example.com'> заранее подключиться к домену
<link rel='dns-prefetch' href='https://example.com'> Заранее подключиться к домену для старых браузеров
не использовать вместе preconnect и dns-prefetch

<link rel='prerender' href='https://example.com/about'> загрузить урл и отобразить на невидимой вкладке

Content Security Policy

Нужен для защиты от XSS
Синтаксис <meta http-equiv='Content-Security-Policy' content="">
Синтаксис для теста <meta http-equiv='Content-Security-Policy-Report-Only' content="">

Ресурсы

default-src: для всего
font-src: шрифты
frame-src: фреймы
img-src: картинки
media-src: видео и аудио
object-src: flash и плагины
script-src: js
style-src: css

Значения

'none' ничего
'self' текущий домен, но не поддомены
'unsafe-inline' позволить встроенный js и css
'unsafe-eval' позволить js eval

Примеры

script-src 'self'
скрипты только текущего домена. встроенные js не работают

script-src 'self' https://jquery.com
скрипты только текущего домена и jquery.com

default-src 'self'; script-src 'self' https://jquery.com
всё только текущего домена. скрипты только текущего домена и jquery.com

style-src 'unsafe-inline' включить встроенный css
script-src 'unsafe-inline' включить встроенный js

<script type='text/javascript' nonce='a1'></script>
<meta http-equiv='Content-Security-Policy' content="script-src 'self' 'nonce-a1';">
включить встроенный js для 1 места

img-src * разрешить картинки отовсюду
img-src data: разрешить картинки внутри data:image base64

Bootstrap

Плагины это модули UMD
Подключить все плагины bootstrap.min.js
Подключить плагины по отдельности js/dist/*.js
Popper.js требуется плагинам dropdowns, popovers, tooltips
Bootstrap 5 без jQuery
Если бутстрап видит объект window.jQuery, и на теге body нет аттрибута data-bs-no-jquery, то jQuery включится автоматически
Плагины работают через HTML аттрибуты
Для 1 элемента использовать 1 набор. Нельзя по 1 кнопке вызвать тултип и попап одновременно

Конструкторы принимают параметры
new bootstrap.Modal(document.getElementById('modal'), {keyboard: false});

Изменить настройки по умолчанию для плагина
bootstrap.Modal.Default.keyboard = false

Узнать версию плагина
bootstrap.Tooltip.VERSION

Установка npm install bootstrap@next
Подключить css @import "~bootstrap/scss/bootstrap";
Подключить js window.bootstrap = require('bootstrap');

Вызов плагина
new window.bootstrap.Toast(document.getElementById('toast')).show();
new window.bootstrap.Modal(document.getElementById('modal')).show();

Подключить CSS

сразу всё
@import "~bootstrap/scss/bootstrap";
@import "../node_modules/bootstrap/scss/bootstrap.scss";

по отдельности
@import "~bootstrap/scss/functions";
@import "~bootstrap/scss/variables";

Подключить JS

сразу всё
import 'bootstrap';

по отдельности
import 'bootstrap/js/dist/util';
import 'bootstrap/js/dist/dropdown';
